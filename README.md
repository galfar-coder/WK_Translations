Translations Repository for White Knuckle
==========================================

This repository acts as the automated upstream storage for the WKTranslator translations hosted on Weblate. 

Contribute to the translations directly at: 
👉 https://translate.galfar.net/projects/white-knuckle-wktranslator/

---

## 💬 Community and Discussion

If you want to discuss translations, suggest new localization features, coordinate with other translators, or report issues, please join our Discord server:

[Join the Discord Server](https://discord.gg/YaQu8KbvwA)

Once you join, look for the **#translations** channel to get started.
